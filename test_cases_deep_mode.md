## Case 2: 机器学习神经网络（高级 · 文本）

**难度**：advanced  
**输入模式**：仅文本  
**测试重点**：复杂概念 + 长生成，触发 JSON 解析压力

**系统提示词**：
```
我想学习机器学习中的神经网络和反向传播算法
```

**补充材料**：

神经网络是一种受人脑结构启发的计算模型，由相互连接的节点（神经元）组成。每个连接都有一个权重，在训练过程中不断调整。前向传播是指输入数据从输入层经过隐藏层到达输出层的过程。反向传播（Backpropagation）是训练神经网络的核心算法，它通过计算损失函数对每个权重的梯度，并使用梯度下降法来更新权重，从而最小化预测误差。常见的激活函数包括 ReLU（f(x)=max(0,x)）、Sigmoid（f(x)=1/(1+e^-x)）和 Tanh。过拟合是指模型在训练数据上表现很好但在新数据上表现差的现象，可以通过 Dropout、L2 正则化和早停法来缓解。

**预期**：研究阶段多次搜索，生成复杂题目。如果 JSON 解析失败，说明 `<think>` 标签未完全修复

## Case 3: 量子计算（进阶级 · 文本 + URL）

**难度**：intermediate  
**输入模式**：文本 + URL  
**测试重点**：URL 抓取 + 文本联合输入

**系统提示词**：
```
我想了解量子计算的基本原理、量子比特和量子纠缠
```

**补充材料**：

量子计算利用量子力学原理来处理信息。与经典比特只能处于 0 或 1 状态不同，量子比特（qubit）可以同时处于 0 和 1 的叠加态。这意味着 n 个量子比特可以同时表示 2^n 种状态。量子纠缠是指两个或多个量子比特之间存在强关联，即使相隔很远，对一个量子的测量会瞬间影响另一个。这些特性使得量子计算机在特定问题上（如大数分解、搜索算法）比经典计算机有指数级加速。

**URL**：https://en.wikipedia.org/wiki/Quantum_computing

**预期**：先抓取 Wikipedia 页面内容，然后结合补充材料一起传给 AI

## Case 4: 区块链共识机制（高级 · 文本）

**难度**：advanced  
**输入模式**：仅文本  
**测试重点**：长生成测试 max_tokens 是否足够（修复后应不截断）

**系统提示词**：
```
我想深入了解区块链的共识机制，包括 PoW、PoS、DPoS 和 PBFT
```

**补充材料**：

共识机制是区块链网络中节点达成一致的方法。工作量证明（PoW）要求节点通过计算复杂的数学问题来获得记账权，比特币使用这种机制，但能耗很高。权益证明（PoS）根据持有代币的数量和时间来选择验证者，以太坊已转向 PoS。委托权益证明（DPoS）由代币持有者选举代表来验证交易，速度更快但更中心化。实用拜占庭容错（PBFT）通过多轮投票达成共识，适用于联盟链。每种机制在安全性、速度和去中心化程度之间有不同的权衡。

**预期**：如果生成阶段超过 8192 tokens，会触发截断错误。修复后（提升到 16384）应成功

## Case 5: 手冲咖啡（入门级 · 文本 + 补充材料）

**难度**：beginner  
**输入模式**：文本 + 短篇补充材料  
**测试重点**：简单输入 + 补充材料混合

**系统提示词**：
```
我想学习如何制作一杯好的手冲咖啡
```

**补充材料**：

手冲咖啡需要控制几个关键参数：水温在 90-96°C 之间（水温过高会过萃取，过低则萃取不足）；研磨度中等偏细（类似砂糖颗粒）；粉水比约为 1:15 到 1:17。注水手法分为闷蒸（bloom）和主注水两个阶段，先用少量水（约粉量的 2 倍）闷蒸 30 秒释放 CO2，然后缓慢画圈注水。总萃取时间约 2-3 分钟。

**预期**：稳定成功，生成 5-8 道题

## Case 6: 人体免疫系统（进阶级 · 文本）

**难度**：intermediate  
**输入模式**：仅文本  
**测试重点**：科学概念 + 术语密度高

**系统提示词**：
```
我想了解人体免疫系统的工作原理，包括先天性和适应性免疫
```

**补充材料**：

免疫系统是人体的防御机制，分为先天性免疫和适应性免疫。先天性免疫是快速响应的第一道防线，包括物理屏障（皮肤、黏膜）、吞噬细胞（巨噬细胞、中性粒细胞）和自然杀伤细胞（NK cells）。适应性免疫则具有特异性和记忆性，主要由 B 淋巴细胞和 T 淋巴细胞介导。B 细胞产生抗体来中和病原体，T 细胞分为辅助 T 细胞（CD4+，协调免疫反应）和细胞毒性 T 细胞（CD8+，杀死感染细胞）。抗原呈递细胞（如树突状细胞）将病原体片段展示给 T 细胞，从而激活适应性免疫。免疫记忆使得疫苗能够长期保护人体。

**预期**：术语密集，研究阶段可能需要多次搜索

## Case 7: 摄影曝光三角（入门级 · 文本 + URL）

**难度**：beginner  
**输入模式**：文本 + URL  
**测试重点**：URL 抓取有效性 + 三个关联概念

**系统提示词**：
```
我想学习摄影中的曝光三角：光圈、快门速度和 ISO
```

**补充材料**：

曝光三角是摄影中控制三个要素的组合：光圈（Aperture）控制镜头进光量，用 f 值表示（f/1.8 是大光圈，f/16 是小光圈），同时影响景深（大光圈背景虚化强）；快门速度（Shutter Speed）控制传感器曝光时间，高速快门（如 1/1000s）冻结运动，慢速快门（如 1/30s）产生运动模糊；ISO 控制传感器感光度，高 ISO（如 3200）在暗光下可用但引入噪点，低 ISO（如 100）画质更清晰。三者需要平衡：大光圈可以配合高速快门，但景深浅；高 ISO 可以在暗光下手持但画质下降。

**URL**：https://en.wikipedia.org/wiki/Exposure_(photography)

**预期**：URL 抓取曝光三角参数，结合补充材料生成题目

## Case 8: 经济学供需关系（进阶级 · 文本）

**难度**：intermediate  
**输入模式**：仅文本  
**测试重点**：社会科学概念 + 抽象理论

**系统提示词**：
```
我想了解经济学中的供需关系、市场均衡和弹性概念
```

**补充材料**：

供需模型是微观经济学的基础。需求定律指出价格上升时需求量下降（需求曲线向右下倾斜），供给定律指出价格上升时供给量上升（供给曲线向右上倾斜）。市场均衡是需求曲线和供给曲线的交点，此时供给量等于需求量。均衡价格和均衡数量由此决定。当外生冲击（如技术进步、政策变化）发生，均衡会移动。弹性衡量一个变量对另一个变量的敏感程度：需求价格弹性 = 需求量变动百分比 / 价格变动百分比。弹性大于 1 为富有弹性（奢侈品），小于 1 为缺乏弹性（必需品）。收入弹性和交叉弹性分别衡量需求对收入变化和相关商品价格变化的反应。

**预期**：概念嵌套多，研究阶段可能需要搜索相关案例

## Case 9: 美拉德反应（入门级 · 文本）

**难度**：beginner  
**输入模式**：仅文本  
**测试重点**：冷门交叉学科（化学+烹饪），验证搜索工具泛化能力

**系统提示词**：
```
我想学习烹饪中的美拉德反应和焦糖化反应的区别
```

**补充材料**：

美拉德反应（Maillard Reaction）是氨基酸与还原糖在加热时发生的一系列复杂反应，产生褐色物质和数百种风味化合物。它在 140-165°C 发生，是烤肉、烤面包和咖啡烘焙时产生香味的关键。焦糖化反应（Caramelization）是糖类在高温下（约 160°C 以上）热分解产生焦糖色和苦甜风味的反应，不涉及氨基酸。两者的主要区别在于：美拉德需要蛋白质/氨基酸参与，焦糖化只需要糖；美拉德产生更复杂的风味，焦糖化风味较单一。实际烹饪中两者常常同时发生，例如煎牛排时表面既有美拉德反应也有少量焦糖化。

**预期**：冷门主题，可能搜索结果较少，验证补充材料的兜底效果

## Case 10: 太阳系行星（进阶级 · 数据密集型）

**难度**：intermediate  
**输入模式**：仅文本  
**测试重点**：数据量大，验证工具循环次数和信息压缩能力

**系统提示词**：
```
我想了解太阳系八大行星的特征、轨道参数和形成理论
```

**补充材料**：

太阳系有八大行星，分为类地行星（水星、金星、地球、火星）和类木行星（木星、土星、天王星、海王星）。水星是最小的行星，轨道离心率最大，没有大气层。金星大小与地球相似，但有浓厚的 CO2 大气层，表面温度高达 465°C（失控温室效应）。地球是已知唯一有液态水和生命的行星。火星有稀薄大气、极地冰冠和古代河流痕迹。木星是质量最大的行星，有著名的大红斑（持续数百年的风暴）。土星有壮观的行星环，由冰块和岩石组成。天王星自转轴倾斜 98°，几乎是"躺着"转的。海王星离太阳最远，风速可达 2100 km/h。太阳系形成于约 46 亿年前的太阳星云坍缩，目前最受支持的理论是星云假说（Nebular Hypothesis）。

**预期**：数据量大，研究阶段可能工具循环到上限。生成阶段可能题目较多
